FAQ

    TTS
    Text-to-Speech

    Why can't I hear pronunciation?

    This app uses your device's built-in text-to-speech (TTS) engine. If the speak button doesn't produce sound, your device likely doesn't have a voice installed for the target language.

    Setup
    By platform

    Android

    1. Open Settings > System > Language & input > Text-to-speech output.
    2. Tap the gear icon next to your TTS engine (usually "Google TTS").
    3. Tap Install voice data and download the language you need (e.g. Japanese, French).
    4. Return to the app and tap the speak button again.

    Tip: If you use Samsung, the path may be Settings > General management > Language > Text-to-speech.

    iPhone / iPad (iOS)

    1. Open Settings > Accessibility > Spoken Content.
    2. Tap Voices.
    3. Select the language (e.g. Japanese, French) and download a voice.
    4. Return to Safari and tap the speak button.

    Windows

    1. Open Settings > Time & Language > Language & region.
    2. Click Add a language and install the language you need.
    3. Make sure to check Text-to-speech in the optional features during install.
    4. Restart your browser and try again.

    macOS

    1. Open System Settings > Accessibility > Spoken Content.
    2. Click System Voice > Manage Voices…
    3. Find the language you need and click the download button.
    4. Restart your browser and try again.

    Linux (Chrome / Chromium)

    Chrome on Linux uses the speech-dispatcher backend. Install additional voices via your package manager: 

    # Debian / Ubuntu
sudo apt install espeak-ng-data

# Fedora
sudo dnf install espeak-ng

# Arch
sudo pacman -S espeak-ng

    Restart Chrome after installing.

    General
    Other questions

    How do I switch languages?

    Tap the flag button in the top-right corner of any page. The app currently supports Japanese and French. Your selection is saved and persists across sessions.

    Is my progress saved?

    Yes. All progress is stored in your browser's local storage. Each language has its own separate progress, so switching languages won't affect your other data.

    Does this app work offline?

    Yes. After the first visit the app is cached by a service worker and works fully offline. TTS may require an internet connection on some devices if the voice data isn't downloaded locally.
